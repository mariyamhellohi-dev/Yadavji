# **App Name**: MatkaMax Dashboard

## Core Features:

- Live Game Status Updates: Display real-time betting statuses (open/closed) with color-coded indicators for each game/bazar.
- Quick Action Buttons: Provide easy access to 'Add Fund,' 'Withdraw,' 'Support,' and 'Download App' actions in a prominent section.
- Real-Time Results: Display winning numbers using result data.
- Side Navigation Menu: Offer a slide-out menu with quick access to key sections like 'Home,' 'Bidding History,' and 'Logout'.


## Style Guidelines:

- Primary color: Bright yellow (#FFC107) to establish the app's identity, drawing from the theme of the online Matka betting application.
- Background color: Light cream (#FFF6D5) to complement the primary color and provide a soft, visually appealing backdrop.
- Accent color: Orange (#FF9800), an analogous color to the primary, offering vibrant visual contrast for key interactive elements and highlighting opportunities. It creates a sense of activity and engagement.
- Font family: 'Poppins', a sans-serif font providing a geometric, modern, and clean appearance suitable for both headers and body text, and readable in both mobile and desktop resolutions. Note: currently only Google Fonts are supported.
- Use a max width of 1200px and a 12-column grid for desktop, ensuring a consistent and responsive layout.
- Use Lucide icons throughout the application to maintain a consistent and modern visual style.
- Subtle press animations for buttons and a smooth slide-in animation for the drawer menu.
